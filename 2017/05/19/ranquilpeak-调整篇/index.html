ranquilpeak 调整篇

Tranquilpeak 主题美观简洁，支持多种样式自定义修改,具体参考官方用户文档 由于Tranquilpeak 和Hexo 存在版本差异和兼容性问题，在配置主题和书写文章时会可能会存在问题

1. 主题语言修改
  • 打开博客_config.yml配置文件修改language: zh-cn

    themes/tranquilpeak/languages
    注意和支持的语言文件名称是否一致区分大小写

2. 修改关于提示页面信息

  • 进入 themes/tranquilpeak/languages 选择语言文件

    你的个人简介 (支持 Markdown 和 HTML 语法) 

    author:
bio: "Markdown/HTML"
#你的工作简介
job: "Markdown/HTML"
